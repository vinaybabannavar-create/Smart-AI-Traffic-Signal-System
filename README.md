# Smart AI Traffic Signal System

AI-driven hybrid traffic optimization system using ML-based congestion prediction and real-time metropolitan forecasting to minimize urban wait times.

## 🚀 Overview

A high-fidelity traffic management platform powered by Random Forest ML models. It visualizes metropolitan traffic flow, predicts congestion patterns, and implements dual-layer optimization (Plan A & B) to streamline urban transport.

## ✨ Key Features

- **Metropolitan Forecast Grid**: Real-time multi-path traffic visualization.
- **ML Congestion Prediction**: Uses Random Forest regression to forecast traffic density.
- **Dual-Layer Optimization**: Implements advanced signal timing strategies.
- **Real-Time Dashboard**: Interactive UI built with React and Tailwind CSS.

## 🛠 Tech Stack

- **Frontend**: React, Vite, Tailwind CSS, Framer Motion, Recharts.
- **Backend**: Python, Flask, Scikit-learn (Random Forest).

## 🚦 Getting Started

### Prerequisites

- Node.js (v18+)
- Python (v3.9+)

### Installation

1. Clone the repository:
   ```bash
   git clone https://github.com/vinaybabannavar-create/Smart-AI-Traffic-Signal-System.git
   cd Smart-AI-Traffic-Signal-System
   ```

2. Install dependencies:
   ```bash
   # Frontend
   npm install

   # Backend
   cd backend
   pip install -r requirements.txt
   ```

3. Run the application:
   ```bash
   # Backend
   python app.py

   # Frontend (in root directory)
   npm run dev
   ```
